# 🎯 Brain-Ring 2026

Система ведения игры «Брейн-ринг» — React + Node.js + Socket.io.

---

## Структура проекта

```
brain-ring/
├── server/
│   ├── server.js          ← Node.js + Socket.io
│   ├── data.json          ← Автосохранение состояния (создаётся при первом запуске)
│   ├── questions.json     ← Пример вопросов
│   └── package.json
└── client/
    ├── src/
    │   ├── context/SocketContext.js   ← Общий стейт через WebSocket
    │   ├── hooks/useAudio.js          ← Звук успеха (Web Audio API)
    │   ├── views/
    │   │   ├── AdminView.jsx          ← /admin — панель ведущего
    │   │   └── PlayerView.jsx         ← /player — экран проектора
    │   └── App.js
    └── package.json
```

---

## Быстрый старт

### 1. Сервер

```bash
cd server
npm install
npm start
# → http://localhost:3001
```

### 2. Клиент (в другом терминале)

```bash
cd client
npm install
npm start
# → http://localhost:3000
```

### 3. Открыть в браузере

| URL | Назначение |
|-----|-----------|
| `http://localhost:3000/admin` | Панель ведущего |
| `http://localhost:3000/player` | Экран игрока (проектор) |

Экран `/player` — открыть в отдельном окне и вынести на проектор.  
При обновлении страницы состояние восстанавливается автоматически.

## Функциональность

### Панель ведущего (`/admin`)
- **📂 Загрузить JSON** — загрузка файла с вопросами
- **← / →** навигация по вопросам (сбрасывает «показать ответ» и блиц)
- **👁 Показать/скрыть ответ** — транслируется на экран игрока с анимацией
- **Редактирование** — изменение текстов и ответов прямо в интерфейсе
- **Попытка 1 / 2** (блиц) — при обеих активных показывает «ВОПРОС СНЯТ» на экране игрока
- **Команды** — добавить, удалить, переименовать (клик по имени), +1 / −1, ручной ввод очков (клик по цифре)
- **Список вопросов** — быстрый переход к любому вопросу
- **Кнопка «🖥 Экран»** — открывает `/player` в новой вкладке

### Экран игрока (`/player`)
- Вопрос на русском (белый, крупный, жирный)
- Вопрос на украинском (голубой, курсив)
- Ответ появляется с fade-in анимацией
- Таблица лидеров вверху с анимацией при изменении очков
- Бегущая строка внизу: `Команда (очки) · Команда (очки)`
- Красный оверлей «ВОПРОС СНЯТ» при блице с 2 попытками
- Звук успеха (Web Audio) при +1 к очкам
